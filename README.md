# FMP Workshop Registration (Light Theme)

## Fix for “no dropdowns”
If you open `index.html` directly, some browsers block `type="module"` scripts.
So:
- Use VS Code “Live Server” OR
- Deploy to GitHub Pages

Dropdowns also have fallback options in HTML now, so you’ll always see something.

## Setup
1) Run `supabase.sql` in Supabase SQL Editor (replace YOUR_ADMIN_EMAIL_HERE)
2) Paste SUPABASE_URL + SUPABASE_ANON_KEY into `config.js`
3) Deploy to GitHub Pages
